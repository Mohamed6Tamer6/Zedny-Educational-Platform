# Scripts Directory

This directory contains utility scripts for database management and administration.

## Available Scripts:

### Database Improvements
- `apply_improvements.py` - Apply database performance improvements
- `verify_improvements.py` - Verify improvements were applied correctly

### Administration
- `manage_admin.py` - Create and manage super admin users

### Development
- `kill_and_run.ps1` - Kill existing server and restart (PowerShell)

## Usage:

```bash
# Apply database improvements
python apply_improvements.py

# Create super admin
python manage_admin.py

# Verify improvements
python verify_improvements.py
```
